# Сенсор Suite — онлайн-демо

Единое тестовое приложение отделов: Документооборот (документы, лицензирование,
проверка специалистов, реестр УЦ), НОК/СРО-калькулятор, Продажи (скрипты, контроль
звонков, RAG-ассистент), Управление (РНП-дашборд), Контрагенты.

Демо-данные обезличены: внутренние скрипты/транскрипты/шаблоны/индексы и любые
персональные данные убраны — оставлены каркас, логика и fallback-наполнение.

Локальная/десктоп-версия (Mac/Windows) с реальными данными и интеграциями
(Google Sheets / amoCRM / SPARK / DaData) поставляется отдельно.
